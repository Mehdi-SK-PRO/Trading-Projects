### 1. Delta Hedging Simulator

Simulation of a dynamic delta-hedging strategy for a vanilla equity call option using Python.  
The project models how a flow trader dynamically hedges a client option position by trading the underlying asset, while tracking daily PnL and option risk sensitivities over time.

#### Outputs
- Daily hedge positions
- Option values and PnL
- Delta, gamma and vega exposure
- Hedging effectiveness analysis
